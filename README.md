# mongodb-ace-theme [![][npm_img]][npm_url]

> MongoDB [ACE Theme](https://cloud9-sdk.readme.io/docs/themes)

## Installation

```
npm install --save mongodb-ace-theme
```

## Usage

```javascript
import 'mongodb-ace-theme';
editor.setTheme("ace/theme/mongodb");
```
